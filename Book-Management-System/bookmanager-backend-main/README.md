# bookmanager-backend

## Install [sdkman](https://sdkman.io/):

```shell
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

## Install Java

```shell
sdk install java 23-open
```

## Install Gradle

```shell
sdk install gradle 8.10
```

## Run the app

```shell
gradle bootRun
```
